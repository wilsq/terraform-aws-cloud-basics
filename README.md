# Terraform AWS Cloud Basics

Basic Terraform project for learning AWS Infrastructure as Code.

## What this project does

- Creates a VPC with a public subnet
- Adds internet gateway and routing
- Deploys an EC2 instance using Amazon Linux 2023
- Installs and starts nginx automatically using user_data

## Region

- eu-north-1 (Stockholm)

## Purpose

This project is created as part of my learning path towards Cloud / DevOps roles.
